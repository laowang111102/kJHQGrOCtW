# 宠物商店管理系统 java1092

## 项目概述

宠物商店管理系统是一款集用户管理、商品管理、公告发布和购物车管理等功能于一体的在线平台。该系统旨在为宠物商店提供便捷高效的运营管理解决方案。

## 技术栈

- **后端**：PHP + MySQL
- **前端**：HTML + CSS + JavaScript

## 功能模块

### 用户模块

- 用户账号的登录与注册

### 管理员模块

- 管理员账号的登录与注册
- 商品管理
- 公告管理

### 商品模块

- 商品选购
- 商品查找

### 购物车模块

- 购物车管理

## 系统角色

- **用户**：可进行商品浏览、购买和购物车管理。
- **管理员**：负责商品和公告的管理，以及用户管理。

## 运行环境

- PHP 7.0 或以上版本
- MySQL 5.6 或以上版本

## 部署步骤

1. 准备运行环境（PHP + MySQL）
2. 创建数据库并导入系统所需数据表
3. 配置系统参数（如数据库连接）
4. 部署后端代码至服务器
5. 部署前端代码至服务器
